# Personal AI Tutor System
Project Overview

The Personal AI Tutor System is an intelligent web-based learning platform designed to support students studying Software Engineering. The system provides personalized learning assistance using artificial intelligence technologies such as chatbots, machine learning models, and automated evaluation.

The platform helps students understand theoretical concepts, assess their knowledge through quizzes, receive automated feedback on descriptive answers, and obtain career recommendations based on their skills and performance.

# Features

- AI Chatbot for answering Software Engineering questions
- Quiz system for knowledge assessment
- Automated theory answer evaluation
- Career prediction and recommendation module
- Student progress tracking and performance analysis
- User authentication and profile management

# Installation and Setup
1. Clone the repository
 - git clone https://github.com/vipusikan/personal-ai-tutor.git
2. Navigate to the project folder
 - cd personal-ai-tutor
3. Install backend dependencies
 - npm install
4. Install frontend dependencies
 - cd client
 - npm install
5. Run the backend server
 - npm start
6. Run the frontend application
 - cd client
 - npm start

The application will run on:
 - http://localhost:3000
